# jsCalculator
A simple JavaScript calculator which supports mouse clicks and keyboard input.
Features include basic arithmetic operations, decimal numbers, AC (all clear), and backspace functionality.

## Features 
- Addition (+), subtraction (-), multiplication (*), and division (/)
- Decimal numbers (.)
- AC button to reset the calculator
- Backspace button to delete the last digit
- Supports keyboard input for numbers and operators
